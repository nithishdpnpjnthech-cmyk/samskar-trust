# Gallery Folder Organization

This folder contains all the gallery images organized by category:

## Folder Structure
```
gallery/
├── training/          # Training program images
├── success/           # Success story images  
├── events/           # Event and celebration images
└── README.md         # This file
```

## How to Add Images

1. **For Training Programs**: Save images in `gallery/training/`
   - Examples: workshops, skill development sessions, vocational training
   
2. **For Success Stories**: Save images in `gallery/success/`  
   - Examples: women entrepreneurs, beneficiary achievements, life transformations
   
3. **For Events**: Save images in `gallery/events/`
   - Examples: award ceremonies, community gatherings, celebrations

## File Naming Convention

Use descriptive names for easier identification:
- `training_session_1.jpg`
- `entrepreneurship_success_story.jpg`
- `award_ceremony_2024.jpg`

## Supported Formats

- **Images**: JPG, PNG, GIF
- **Videos**: MP4, WebM
- **Max Size**: 10MB per file

## Website Integration

When you upload images through the website's upload feature, they will be automatically categorized into the appropriate folders based on your selection.
